# TaskManager CLI
![CI](https://github.com/DeyversonLima/task-manager-cli/actions/workflows/ci.yml/badge.svg)

## Descrição
O TaskManager CLI é uma aplicação simples de linha de comando desenvolvida para ajudar usuários a organizarem suas tarefas do dia a dia, reduzindo esquecimentos e melhorando a produtividade.

## Problema
Muitas pessoas enfrentam dificuldades em gerenciar suas tarefas diárias, o que pode gerar desorganização e perda de prazos importantes.

## Solução
A aplicação permite cadastrar, listar, concluir e remover tarefas de forma simples e rápida via terminal.

## Público-alvo
- Estudantes
- Profissionais
- Pessoas com rotina corrida

## Funcionalidades
- Adicionar tarefas
- Listar tarefas
- Marcar tarefas como concluídas
- Remover tarefas

## Tecnologias utilizadas
- Python 3
- Pytest
- Flake8
- GitHub Actions
